
# vue-qq-music


> 项目来源于慕课网，Vue.js 高级实战——音乐 App 课程。

欢迎`issue`，`pr`，`star` or `follow`！


## 工具&技能

`Vue 2.x` + `Vuex 2.x`+ `Vue Router` 技术框架

`axios` + `jsonp` 服务端通讯

`HTML`  + `Stylus` + `ES6+` 技术语言

`webpack 2.0` 构建工具

`better-scroll` 移动端滚动库


## 实现功能

- 歌单推荐页面

- 歌单详情页

- 歌手页面

- 歌手详情页

- 排行页面

- 排行详情页

- 搜索页面

- 歌曲列表页

- 播放器页面
